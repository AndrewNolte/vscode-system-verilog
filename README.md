# Verilog HDL support for VSCode
Verilog HDL support based on [https://github.com/textmate/verilog.tmbundle](https://github.com/textmate/verilog.tmbundle) TextMate package.

## Features
### Done
- Syntax highlighting for `.v` `.vh` files
- Simple Snippets
   * `module`
   * `always`
   * `case`
   * `for`
   * `while`
   * `function`
   * testbench template
   * etc...
